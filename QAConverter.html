QA Sheet Generator!

Instructions

  1. Navigate to the View/Edit Invoice page in Quickbooks.
  2. Right click anywhere in the background and click "Inspect"
  3. In the new window that appears, click the first line of computer code and press CTRL+C (or CMD+C) to copy it.
  4. Paste the code below and click "Go!"




You can save the results as a PDF or send it to a printer by simply pressing CTRL+P (or CMD+P).
To start over, simply refresh the page or open the program again in a new tab!
